navigator--tiene las funciones del navegador, permite acceder al sistema operativo como gps o disco duro
window--maneja las pestañas
docuemnt--contiene todos los elemnetos visuales dentro de la página web

window es un objeto y location es una variable o atributo o propiedad del objeto por eso no tiene () su valor se puede poner luego de un = o guardarse en una variable --var--
SRC es un atributo de la etiqueta script para traer un script de otro lado
